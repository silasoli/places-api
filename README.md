## Description

Places API that counts with CRUD of places, users create route and authentication route.

## Installation

```bash
$ npm install
```
## Before running the app
```bash
# 1. RUN npm install;
# 2. RUN docker compose up;
# 3. Fill in all environment variables;
# 4. Make sure the database is created;
# 5. Run all migrations;
# 6. Reload the app container;
# 7. Open http://localhost:3000/docs in your browser.
```
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
## Database Commands

```bash
# run migrations
$ npm run migration:run
```
## Test commands

```bash
# run coverage
$ npm run test:cov
```
